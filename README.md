# BioTrack

A scientific association wants to track the movements of wild animals fitted with GPS tags—such as sharks, albatrosses, turtles, wolves, and eagles. They have engaged our company to develop a mapping application that visualizes this data on an interactive map.
As part of the development team, our mission is to create the interactive map that displays the animals' GPS positions. The rest of the team handles integration into the full system, including menus, email alerts, user management, and other platform features.

## Resources

- [Leaflet 1.9.4](https://leafletjs.com/) — interactive map rendering
- [Leaflet.locatecontrol](https://github.com/domoritz/leaflet-locatecontrol) — geolocation control
- [CartoDB Basemaps](https://carto.com/basemaps/) — dark mode tile layer
- [OpenStreetMap](https://www.openstreetmap.org/) — default tile layer
- Data: GPS tracking records from [OCEARCH](https://www.ocearch.org/) shark tracking program
